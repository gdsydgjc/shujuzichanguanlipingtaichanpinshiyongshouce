## 功能亮点

- **全方位元数据整合**：轻松接入关系数据库、客户定制化元数据等多种数据源，实现元数据的一站式管理。

  - **统一知识库存储**：所有采集的元数据经过系统整理后，存储在统一的知识库中，为高级数据分析和服务提供坚实基础。
